# 🏮 NFC Digital Capsule | 交互式数字胶囊

> 连接物理与数字世界的时空胶囊。
> A Cyberpunk gift linking physical artifacts to digital memories.

## 📖 项目简介 (Introduction)

这是一个运行在 **GitHub Pages** 上的静态网页项目，配合 **NFC (近场通信)** 实体挂件使用。它不是一个普通的网站，而是一个依附于物理实体的“数字操作系统”。

通过扫描 PCB 挂件上的 NFC 芯片，用户将进入一个集 **视觉祝福**、**音乐体验** 与 **私密记忆存储** 于一体的中心化入口 (Hub)。

## 🧩 功能模块 (Modules)

### 1. 🎛️ The HUB (入口)
极客风格的主菜单，连接所有子功能的中心枢纽。
- **左键 (...)**：预留的未来扩展接口（彩蛋位）。
- **中键 (ECHO)**：内向的记忆胶囊。
- **右键 (祝福)**：外向的视听展示。

### 2. 🎉 For Ya (祝福特效)
- **核心体验**：满屏的新年祝福弹幕 + 旋转唱片动画。
- **背景音乐**：自动播放《给你呀》 (需用户交互触发)。
- **交互**：点击中心的“福”字 PCB 即可开启视听盛宴。

### 3. 🌑 ECHO (回声系统)
> **"写下一段你想留下的回声，写下后不可更改，只保留10次记录，且行且珍惜。"**

ECHO 是一个基于 **LocalStorage (本地存储)** 的极简留言系统。它是本项目最核心的哲学表达。

#### 🛡️ 隐私与数据安全声明 (Privacy & Philosophy)
**ECHO 没有任何云端数据库，不进行任何网络上传。**

* **数据归属**：所有的文字记录都**只存储在扫描该 NFC 的那台手机浏览器中**。
* **物理绑定**：记忆不是跟账号绑定的，而是跟“这个物体 + 这部手机”绑定的。这意味着，只有持有这部手机的人，再次扫描这个挂件，才能看到自己曾经留下的回声。
* **绝对私密**：即便是项目的开发者（我），也无法看到你在 ECHO 中写下的任何内容。它是完全属于你自己的数字秘密。

#### ⏳ 稀缺性设计 (Scarcity)
* **不可逆**：写下即刻铸造，不支持修改或删除。
* **有限性**：系统仅允许写入 **10条** 记录。当记忆填满，ECHO 将永久封存，成为只读的历史。

## 📱 关于 NFC 兼容性 (Why Android Only?)

本项目推荐使用 **Android (安卓)** 设备进行完整体验，尤其是写入操作。

虽然现代 iOS 设备也支持 NFC，但存在以下系统级限制，使得 Android 成为更佳的选择：

1.  **协议开放性**：Android 拥有完全开放的 NFC 读写权限。无论是在主屏幕还是任何 App 中，安卓都能灵敏地捕捉到 NFC 信号并直接唤起浏览器。
2.  **后台读取能力**：
    * **Android**：几乎所有带有 NFC 功能的安卓手机（无论新旧）都支持后台读取标签。
    * **iOS**：只有 iPhone XS 及以上机型才支持“后台标签读取”。旧款 iPhone 必须打开专门的 App 才能扫描，破坏了“一触即达”的惊喜感。
3.  **抗干扰性**：iOS 的 NFC 模块常驻 Apple Pay（钱包），在扫描非标准或未加密的 DIY 标签时，iPhone 经常会优先弹出“钱包”界面而不是打开网页，严重影响体验流畅度。

**结论**：为了获得最纯粹、无缝的“赛博朋克”交互体验，请使用 Android 设备触碰挂件。

## 🛠️ 使用说明 (How to Use)

1.  **准备**：
    * 一个 NTAG215 芯片 (或 PCB 线圈)。
    * 一部支持 NFC 的 Android 手机。
    * 安装 "NFC Tools" (或类似 App)。
2.  **写入**：
    * 在 App 中选择 "Write" -> "Add record" -> "URL / URI"。
    * 输入本项目的主页地址：`https://你的用户名.github.io/你的仓库名/`
    * 点击 "Write" 并将手机靠近芯片。
3.  **体验**：
    * 无需打开任何 App，解锁手机，将芯片靠近手机背部 NFC 区域（通常在摄像头附近）。
    * 系统将自动弹出通知或直接跳转至 `NFC HUB` 页面。
